This project is the result of transitioning from a legacy Electronic Disease Surveillance System (EDSS) to a brand new one. We are focusing primarily on migration of data. The legacy system is Maven and the new system is Origami.

Here are the preliminary steps:
1. Identify base case data elements from the legacy system
2. Identify if there is an exact or close match with a corresponding data element in the new system
3. Confirm that the matching elements have identical value sets

This process will have to be repeated for every disease or condition.
